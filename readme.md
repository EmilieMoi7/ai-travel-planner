# AI Travel Planner

## Présentation

AI Travel Planner est une plateforme d'aide à la décision basée sur l'intelligence artificielle pour le secteur du voyage.

L'objectif est de développer une application capable d'accompagner :

- les voyageurs dans la préparation de leur séjour ;
- les professionnels de l'hôtellerie dans l'analyse des réservations.

Ce projet est réalisé dans le cadre de la constitution d'un portfolio en Data Science, Machine Learning et Intelligence Artificielle.

---

## Fonctionnalités

### Voyageur

- Estimation du prix d'un vol
- Estimation du budget d'un voyage
- Recommandation de destination

### Professionnel

- Analyse du risque d'annulation des réservations
